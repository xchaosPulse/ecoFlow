# Contributing to the Iran Open Economy Dashboard

We welcome contributions to our data! Please follow these steps to contribute new data or update existing data.

## How to Contribute

1.  **Fork the repository.**
2.  **Create a new branch** for your changes.
3.  **Add or update data** in the `/data` directory. Please ensure the data is in JSON format and follows the existing schema.
4.  **Create a pull request** with a clear description of your changes.

## Data Schema

Please refer to the existing files in the `/data` directory for the data schema.

## Code of Conduct

Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms.
